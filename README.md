# 农产品销售系统 python308

## 项目概述

农产品销售系统是一款基于Python语言开发的应用，旨在提供农产品在线销售平台，实现用户、商家和管理员之间的交互与操作。

## 技术栈

- Python 3.7
- Django 框架
- MySQL 5.7
- Vue.js

## 功能模块

### 用户功能

- 首页浏览
- 个人中心管理
  - 修改密码
  - 我的订单
  - 我的地址
  - 我的收藏

### 管理员功能

- 个人中心管理
- 用户管理
- 商家管理
- 产品类型管理
- 农产品管理
- 系统管理
- 订单管理

### 商家功能

- 系统首页浏览
- 个人中心管理
- 农产品管理
- 订单管理

## 系统角色

- 用户：浏览和购买农产品，管理个人信息等。
- 管理员：系统维护，用户与商家管理，农产品和订单管理等。
- 商家：发布和更新农产品信息，处理订单。

## 运行环境

- 服务器：支持Python和Django环境的服务器
- 数据库：MySQL 5.7
- 前端：支持HTML5和JavaScript的浏览器

## 部署步骤

1. 搭建Python 3.7及Django框架环境
2. 配置MySQL 5.7数据库
3. 部署后端代码
4. 部署前端Vue.js代码
